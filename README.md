## J. M. Rajan Ministries Website

React + Vite + TypeScript single-page site with banner, vision, mission, and social links.

### Getting Started

1) Install dependencies

```bash
pnpm install
# or
npm install
# or
yarn
```

2) Run the dev server

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```

3) Build for production

```bash
pnpm build
# or
npm run build
# or
yarn build
```

4) Preview the production build

```bash
pnpm preview
# or
npm run preview
# or
yarn preview
```

### Project Structure

- `src/components` — UI components for header, banner, sections, and footer
- `src/styles.css` — Theme and layout styles
- `public/logo.png` — Place your ministry logo here (optional)
- `public/profile.jpg` — Place a profile image if desired (optional)
